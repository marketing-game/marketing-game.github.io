别再熬夜分析竞品！用 ChatGPT Task 实现每天自动监控和分析 （附Prompt）

发表于2025-10-22|更新于2025-10-22|ChatGPT
|阅读量:

0. 熬夜到两点，只为赶一份竞品报告？

你是否经常遇到下面的场景：

晚上 11 点刚合上电脑，手机“叮——”地跳出老板的消息：

“明早例会前，麻烦把竞品 A、B、C 最新功能对比和舆情走势整理一下。”

心跳瞬间飙到 180——打开官网、微博、公众号、头条、抖音、小红书等一通瞎看，表格刚拉好，社区又冒出新帖子；爬完 A 的更新日志，B 的产品说明书还没动；眼看凌晨两点，咖啡已经续到第三杯。

可怕的是，这样的“临时炸弹”几乎每天都在重演：信息量爆炸，时间永远不够，分析永远赶不上变动。

如果经常遇到，那么恭喜你，你的救世主—— ChatGPT Task 来了，真正的天选打工人，先天牛马圣体！！

1. 牛马 ChatGPT Task 是什么？

一句话先说结论：

Task = ChatGPT 里的自动化小助手，能在指定的时间，按照你的要求进行执行任务，并将执行结果按你的要求进行产出，邮件或手机通知到你。——做到这一切，有嘴就行。

凡事有交代，件件有着落，事事有回应——合格的牛马。

注：ChatGPT Task 需要 Plus 及以上会员才能开启，有升级需求的朋友可以看这里：​**https://gpt101.org**​ ，我自己写的网站。

1.1 牛马ChatGPT Task上岗前的“培训”

现在，我们准备让牛马 ChatGPT Task 上岗，为了有一份高质量的产出，还需要对它进行一些基础培训，主要内容如下：

  1. 时间：比如每天早上 8 点
  2. 分析对象：比如产品A、B、C
  3. 分析范围：比如官网、微博、公众号、投标、抖音、小红书
  4. 分析内容：比如产品 A、B、C 的最新动态，产品对比，融资进展，重大新闻，舆论情况等等
  5. 产出内容：比如产出一份分析报告，报告格式是xxx

1.2 牛马 ChatGPT Task岗前演练

培训完成之后， 就可以对牛马 ChatGPT Task进行岗前演练了。示例 Prompt 如下：

每天早上 8 点，给我一份竞品分析报告，具体要求如下：

竞品分析对象：ChatGPT、Claude、Gemini

信息源：官网、X、Reddit、YouTube、Facebook、微博、头条、抖音、小红书、热门科技自媒体等等

分析内容（近一个月）：技术进展，产品动态，定价策略，运营情况，公司新闻，融资进展，舆论热点等

分析报告：两部分，第一部分按照分析维度，产出分析结果，第二部分形式为表格，行-维度，列-公司，展示分析概要结果

模型选择 O3或者 GPT-5 thinking，效果更佳。

接下来是试运行的效果，你可以直接在这个对话框中和他说：

是不是效果还行？

这只是一个案例，上面那个 Prompt 写的比较简单，你可以根据自己的需求自行设计 Prompt。

3. 拓展场景

稍微一想，凡是需要重复执行、持续跟进的任务，都可以用 ChatGPT Task 实现：

  • 竞品分析
  • 热点分析
  • 最新动态
  • 舆情监控
  • 学术期刊跟进
  • 等等

基本 ChatGPT 能干的事情，Task 中都能干，包括数据分析这些。剩下的就看大家自由发挥了。
