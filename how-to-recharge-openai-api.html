最近许多用户在完成OpenAI账户注册后发现，原本应提供的5美元免费API调用额度不再存在。当尝试使用API密钥时，系统会提示“配额不足”（insufficient quota）。这一变化源自OpenAI自3月21日起对新账户政策的调整，取消了赠送5美元信用额度的优惠。

​​

OpenAI API在最初推出时，为了吸引用户，提供了18美元的免费额度，有效期为3个月。

随后，这一额度被缩减至5美元，并且维持了一段时间。

然而，由于部分用户利用这一免费额度进行不当得利，注册成本相对较低的账户以换取价值较高的API额度，OpenAI决定取消这一政策。
